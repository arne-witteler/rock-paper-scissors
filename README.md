# Rock, Paper, Scissors

A minimalist, modern, and fully responsive "Rock, Paper, Scissors" game built for the browser. Designed with a **Mobile-First** approach, the application features a clean, timeless black-and-white interface.

The game is played in a "Best of Five" format—the first to win 3 rounds against the CPU takes the match.

## Features

- **Mobile-First Design:** Fully optimized for mobile screens with finger-friendly touch targets and compact layouts, while scaling gracefully to large desktop monitors.
- **BEM Architecture:** Built entirely using the strict *Block-Element-Modifier* (BEM) naming convention, ensuring highly clean and maintainable CSS.
- **Inline SVGs:** Modern line-art hand icons that scale flawlessly and adapt dynamically to hover/active button states using `currentColor`.
- **Clean JavaScript Separation:** Structured with semantic HTML5 `data-*` attributes and dedicated `<span>` tags for text nodes to keep UI and gameplay logic strictly separate.
